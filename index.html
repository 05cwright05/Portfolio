WCSC File Management System

Web‎Dev
Presentation Collaboration

Before developing my file management appliaction, Warrick County School Corporation's secretarial office was 100% dependent on physical filing cabinets. I developed a file upload and download interface including an embeded emailing system. This system allows secretaries to easily send necessary documents to employees. Protects confidential information, by storing sorted digital copies.

  • Engineered an intricate file-saving structure
  • Incorporated user-driven input for design.
  • Utilized the WPF framework
  • Picked up .xaml to design UI
  • Leveraged powerful libraries to display PDFs
  • Employed C# to craft and optimize data algorithms
